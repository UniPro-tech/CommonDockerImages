# UniProject Common Docker Images

[![Autometically Update](https://github.com/UniPro-tech/CommonDockerImages/actions/workflows/cron.yml/badge.svg)](https://github.com/UniPro-tech/CommonDockerImages/actions/workflows/cron.yml)

This is the Docker Images commonly used by UniProject.
These images are updating autometically.

These are released under the Apache 2.0 License, a liberal Open Source license, similar to the BSD or MIT licenses.

## Images Details

### Postgre SQL

**Official image: [postgres](https://hub.docker.com/_/postgres)**

This image shows the addition of a Japanese language plugin.

#### Added Plugins

- [pgbigm/pg_bigm](https://github.com/pgbigm/pg_bigm) - Provides full text search capability

## How to update

You **DON'T** have to update manually.
These are updated by a GitHub workflow cron job every day.

## How to use

Everyone use these images with ghcr.io.

Please check [list of GitHub packages](https://github.com/orgs/UniPro-tech/packages?repo_name=CommonDockerImages).
